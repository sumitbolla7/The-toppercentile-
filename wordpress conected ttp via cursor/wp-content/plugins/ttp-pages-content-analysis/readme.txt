=== TTP Pages & Content Analysis ===
Contributors: topercentile
Requires at least: 5.8
Tested up to: 6.8
Stable tag: 2.0.0

Unified analytics hub for thetoppercentile.co.in:

* Overview — Search Console + Hotjar status + page opens
* Search — GSC-style performance via Google Site Kit (query filter e.g. "ttp mocks")
* Behavior — Hotjar heatmaps, recordings, funnels (links + optional tracking)
* Page opens — per-URL log, live feed, CSV export
* Content audit — SEO checks per page/post/product

Works alongside Google Site Kit and Hotjar (does not replace them).

== Installation ==

Upload wp-content/plugins/ttp-pages-content-analysis/ and activate.
Connect Site Kit for search data. Configure Hotjar Site ID in Settings or use the Hotjar plugin.
